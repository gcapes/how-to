# Python classes

- `@static method` - attach a function to a class, without the self variable
- `class_method` creates another constructor type
- `__method_name` is a class local reference to a method to allow subclasses to override methods
- `@property` converts dotted access to method calls (getter)
- `@<attribute>.setter` is a setter decorator
- *helper* methods or *private* methods start with a leading underscore and are intended only to be called from within the class itself.
- `Self` type is a hint used to annotate a method which returns an instance of its class.
